# 📋 Automated Attendance Extraction and Report Generation System

An automation-based solution that streamlines attendance management in educational institutions using UiPath Studio and Python.

🔗 **Project Report:** See `Project_Report.pdf` in this repository

---

## 📌 Overview

Manual attendance tracking in educational institutions is time-consuming and error-prone. This system automates the entire process — from data extraction and validation to report generation and notification — eliminating manual effort and improving accuracy.

---

## 🚀 Features

- 📥 **Automated Data Extraction** from Excel sheets, ERP systems, and biometric devices
- ✅ **Data Validation** — detects missing, duplicate, or incorrect entries
- 📊 **Attendance Percentage Calculation** with threshold-based low attendance detection
- 📄 **Structured Report Generation** in PDF and Excel formats (daily, weekly, monthly)
- 📲 **Automated Notifications** sent to admin and management
- 🔒 **Error Handling & Logging** for reliability and traceability
- ⚡ **Scalable** — handles large datasets without performance degradation

## 🔄 How It Works

1. **Data Collection** — Attendance records extracted from Excel, ERP, or biometric sources
2. **Data Validation** — Records checked for missing or duplicate entries
3. **Attendance Processing** — Percentages calculated; low attendance students flagged
4. **Report Generation** — Structured reports created in PDF/Excel format
5. **Notification & Export** — Reports distributed to admin and management automatically
